Digital Image Processing II/Advanced Image Processing

Wed 3:50-6:40

 Instructors: Gene Gindi, Jerome Liang, Dimitris Samaras

 

This course will cover modern approaches to image processing, statistical image formation and image models, image restoration, reconstruction and segmentation, applications to medical imaging.

Appropriate background: CSE527 or ESE558, or linear algebra, calculus, basic probability, programming. Please consult instructors
