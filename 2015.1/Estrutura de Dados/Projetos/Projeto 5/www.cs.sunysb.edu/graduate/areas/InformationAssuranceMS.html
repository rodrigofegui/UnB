Information Assurance Specialization at the M.S. Level



New: Attractive Scholarships Available for Specialization in IA

In order to graduate with a M.S. degree with specialization in Information Assurance, a student must complete the following core, elective, and project requirements.

Core courses

CSE 508 Network Security 3 credits
CSE 509 Computer System Security 3 credits
CSE 6XX CSE 669, 689, 608, 629, 635 or
other 600-level course in IA 		1 to 3 credits

Electives:

At least two of the following courses (6 credits)

CSE 506 Operating Systems
CSE515 Transaction Processing
CSE532 Advanced Database Systems
CSE533 Network Programming
CSE534 Fundamentals of Network
CSE608 Advanced Computer Security
CSE629 Wireless LANs and Applications
CSE635 Asynchronous Systems
AMS 570 Mathematical Statistics
EST 530 Internet Electronic Commerce
EST 582 Systems Approach to Human-Machine Systems
MGT 520 Ethics in Management
AMS 597 Statistical computing
MAT 516 Probability and Statistics
AMS 572 Data Analysis
AMS 623 Topics in Systems and Control Theory
MGT 553 Simulation and Modeling
MGT 550 Operations Research
ECO 604 Game Theory


Project Requirement

For non-thesis option, the student should do a project (CSE 523/524) in the area of information assurance. For thesis option, the student should do a thesis on an information assurance topic.
