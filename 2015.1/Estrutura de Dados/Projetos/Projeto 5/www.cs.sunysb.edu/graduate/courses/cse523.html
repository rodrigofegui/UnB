Course CSE523
Title Introduction to Software Engineering and Project Plan
Description A project in programming or digital system design that will extend over two consecutive semesters. The student starts the project in one semester by registering for CSE 523 and completes the project in a following semester by registering for CSE 524. Before the deadline date designated by the course instructor, the student will prepare a one to two page description of the work that is expected to be completed during the two-semester sequence. This description, reviewed and approved by the student's advisor, will reside in the student's file. Performance in completing the course requirements will be evaluated with reference to the implied promise contained. Amendments to the project description must be approved by the advisor. This course is graded separately from CSE 524.
Credit Information 3 - credits; Repetitive
