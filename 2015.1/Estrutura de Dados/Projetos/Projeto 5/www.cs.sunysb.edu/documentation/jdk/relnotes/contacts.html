Contacting Java Software

 Documentation Contents

The Java 2 SDK, Standard Edition is a product of Sun Microsystems, Inc. The Java 2 SDK is developed by Java Software, a part of Sun Microsystems.

Java Software welcomes your comments and suggestions. You can contact us through email, or you can file a bug report. But please remember that Java Software does not offer free technical support. If you need help, try the Java Software Assistance Page, where you'll find links to FAQs, online tutorials, and many other resources.

Send Us Email

Feel free to send email directly to Java Software engineering teams using:

Java Software email address list
If you're not sure which team to contact, browse the Guide to Features and look for a "Comments" email link at the bottom of the main page for each feature. Send general comments to jdk-comments@java.sun.com.

Please file documentation bugs as described below. For other general comments about the documentation not targeted to any particular engineering team, please send email to docs@java.sun.com for content including the API specification, or docwebmaster@java.sun.com for web page problems and broken links.

File a Bug Report

You can use our bug tracking system to report bugs or request new features. Use our Bug Report Form.

Download/Installation Problems?

If you're having problems downloading or installing the Java 2 SDK or Java 2 Runtime Environment, tell us about them on our Web Site Feedback Form.
